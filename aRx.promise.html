aRx.promise¶

class aRx.promise.Promise(awaitable: Union[Awaitable[K], Coroutine[[Any, Any], K], None] = None, **kwargs)[source]¶

Bases: aRx.abstract.promise.Promise

Promise implementation that maintains the callback queue using Coroutine.

See: Promise for more information on the Promise abstract interface.

catch(on_reject: Callable[Exception, Union[L, Awaitable[L]]]) → aRx.promise.ChainPromise[K, L][source]¶

Concrete implementation that wraps the received callback on a Coroutine. The Coroutine will await the promise resolution and, if a exception is raised, it will call the callback with the promise exception.

See: catch() for more information.

lastly(on_resolved: Callable[Any]) → aRx.promise.ChainPromise[K, K][source]¶

Concrete implementation that wraps the received callback on a Coroutine. The Coroutine will await the promise resolution and call the callback.

See: lastly() for more information.

then(on_fulfilled: Callable[K, Union[L, Awaitable[L]]]) → aRx.promise.ChainPromise[K, L][source]¶

Concrete implementation that wraps the received callback on a Coroutine. The Coroutine will await the promise resolution and, if no exception is raised, it will call the callback with the promise result.

See: then() for more information.
